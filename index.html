ITC PROJECT

by Kenmario O. Noschese
Conversion Factorial, Sum and Average Calculator Income Tax Calculator Payroll Calculator

CS 1130 - Introduction to Computing

This site is a demonstration of my growing knowledge and skills in web development, showcasing my work with HTML, CSS, and JavaScript. Designed as the final ITC project, the website highlights key features such as interactive elements, responsive design, and dynamic functionality.

The website has 4 parts; convertion, factorial and sum and average, income calculator, and a simple payroll. The convertion asks for user input, and returns the appropriate conversion value based on the selection. The factorial and sum asks for user input and returns the factorial or the sum and after "key up". The income calculator asks for user input and returns the income tax based on several criterias and lastly there is the simple payroll.
